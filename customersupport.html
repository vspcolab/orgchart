📞 Y-Diet – Office & Customer Support Dashboard

👩‍💻 Role: Customer Support Executive / Office Coordinator

Acts as the central bridge between customers and the Y-Diet system. Handles inquiries, order status updates, subscription management, complaints, feedback collection, and supports internal coordination between departments.

🗂️ Daily Tasks

Task ⏱ Time Description
Respond to incoming WhatsApp/calls All day Assist with order tracking, kit info, basic AVAV questions
Confirm new kit orders & dispatch info Morning Double-check names, address, phone number, and user preferences
Log complaints or suggestions Ongoing Enter issues into the feedback tracker for analysis
Alert kitchen or dispatch if an issue arises Immediately Report wrong kit, delay, or special handling request
Record feedback from delivered users Evening Call or send a form to 5–10 users per day post-delivery

🗂️ Weekly Responsibilities

Task When Description
Send subscription renewal reminders Monday Notify users with 1–2 kits left in their cycle
Prepare feedback report Friday Highlight top praise, recurring complaints, and kit suggestions
Clean user contact list Wednesday Remove duplicates, update preferred language, and tag users
Share support highlights Weekly Share shoutouts, issues, and resolutions with internal teams

📆 Monthly Duties

Task Description
Support audit Ensure all queries are resolved within SLA time
Analyze issue trends Report which kits or processes need improvement
Customer Experience Score Create a dashboard from feedback and satisfaction ratings
Coordinate wellness webinar or offline event Host a monthly user meetup or wellness Q&A (if needed)

🧠 Best Practices

  • Reply with care, calm, and empathy — Y-Diet is not just food, it's a transformation journey
  • Personalize responses (e.g., “Hello, Mr. Ramesh — hope your Detox Kit worked well!”)
  • Confirm user AVAV preferences before forwarding issues
  • Track issue resolution time (goal: < 1 hour for common queries)
  • Update internal teams with color-coded tags:
    • 🔴 Urgent
    • 🟡 Info only
    • 🟢 Resolved
    • 🧘 Suggestion
