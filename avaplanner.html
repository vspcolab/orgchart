🧘‍♀️ Y-Diet – AVAV Planner Dashboard

👤 Role: AVAV Planner

Translates user data from daily AVAV logs (Ahāra, Vihāra, Āchāra, Vichāra) into dynamic, personalized kit recommendations. Helps align each user’s food plan with their physical, emotional, behavioral, and spiritual goals.

🗂️ Daily Tasks

Task ⏱ Time Description
Review new AVAV entries Morning Scan all submitted logs (telebot/web) for the past 24 hrs
Identify score deviations Midday Spot major changes (e.g., low Ahāra, high Vichāra)
Recommend updated kit types Afternoon Assign suitable kit category (e.g., Detox, Mind-Calm, Energy Boost)
Sync with Nutritionist Afternoon Verify nutrient compatibility of suggested plan
Trigger kit update Before Dispatch Tag user in database for upcoming cycle change

🗂️ Weekly Responsibilities

Task When Description
User trend report Every Monday Submit insights to leadership on shifting user profiles
Coordinate with Packaging & Dispatch Wednesday Share user-specific kit preferences (e.g., no garlic, gluten-free)
Update AVAV Scoreboard Friday Refresh user leaderboard/streaks for gamification
Attend AVAV Huddle (team) Weekly Collaborate on spiritual alignment with meal journeys

📆 Monthly Duties

Item Goal
Audit personalization outcomes Assess how AVAV plan changes impacted user behavior
Refine kit-mapping algorithm Improve logic linking scores to kit variants
Suggest new kit ideas Based on trends (e.g., low Vichāra + emotional stress)
Collaborate with AI/Tech team Review n8n automation triggers and flag exceptions

🧰 Tools You Use

n8n Workflows Trigger automation → inventory & packaging alerts

🧠 Best Practices

  • Always prioritize consistency of user AVAV logs over one-off spikes
  • Engage with top users through WhatsApp or Email nudges for deeper insight
  • Flag “chronic misalignments” (e.g., low Vichāra for 7+ days) to Coach or Customer Care
  • Use Sankalpa history to understand intent before changing kits
  • Ensure AVAV-aligned kits respect dietary preferences (Satvic, Jain, Vegan)
